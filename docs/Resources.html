How to use this page
On this page, we provide pipelines from major consortia such as ENCODE, Cistrome and Genomic Data Commons (GDC). The pipelines are provided as simplified versions of the original pipelines which are accessible via the website or paper cited for each pipeline. The simplification is just to keep the style consistent with our pipelines. A list of pipelines is shown upon clicking on a consortium button. We have provided only one pipeline for Cistrome, therefore the pipeline is directly provided upon selecting Cistrome. For other consortia, the user can select one of several pipelines to visualize an interactive pipeline in a flowchart style. Upon clicking each step in a pipeline, various software are listed with hyperlinks to their websites.
